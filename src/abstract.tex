\chapter*{Abstract\markboth{Abstract}{}}
\addcontentsline{toc}{chapter}{Abstract}

    This paper documents the research, the author has done on Virtual Library
    Operating Systems and if they are useful for the cloud at the Hochschule
    Furtwangen University.

    The research was done with OSv, A Unikernel operating system for KVM, Xen,
    VMware and Amazon EC2. It explains how to setup a Fedora machine with KVM
    and how to install OSv from source with or without special packages
    included.

    At the end of the document, the author expresses his opinion if OSv is ready
    for production use in the environment of Hochschule Furtwangen University.
