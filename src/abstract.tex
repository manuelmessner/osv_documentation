\chapter*{Abstract\markboth{Abstract}{}}
\addcontentsline{toc}{chapter}{Abstract}

    This paper documents the research, the authors have done on Virtual Library
    Operating Systems and whether they are useful for the cloud at the \HFU.

    The research was done with OSv, a unikernel operating system for KVM, Xen,
    VMware and Amazon EC2. It explains how to setup a Fedora machine with KVM
    and how to install OSv from source without special packages included.

    At the end of the document, the authors express their opinion whether OSv is
    ready for production use in the environment of \HFU.
